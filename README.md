#  Portfolio Website

##  Description
This is a personal portfolio website developed using **HTML** and **CSS** to showcase my skills, projects, and contact information. The website is designed with a clean layout and responsive design to ensure a good user experience across different devices.

---

##  Objective
The objective of this project is to create a well-structured portfolio website that demonstrates my understanding of web design, layout structuring, and styling using CSS.

---

##  Technologies Used
- HTML5
- CSS3

---

##  Website Structure
The website consists of the following sections:

- **Home** – Introduction and basic details
- **About** – Information about my background and interests
- **Skills** – List of technical skills
- **Projects** – Showcase of projects with descriptions and links
- **Contact** – Contact form and social media links

---

##  Features
- Responsive design using media queries
- Navigation menu for easy access to sections
- Project cards with images and links
- Hover effects and transitions
- Smooth scrolling
- Clean and modern UI design
